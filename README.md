# Sudoku

A Sudoku game built with vanilla TypeScript and object-oriented programming, bundled with Vite.

🎮 [Play it here](#) <!-- update with Vercel URL after deploy -->

## Features

- Puzzle generation with guaranteed unique solution
- Error tracking (3 mistakes allowed)
- Undo last move
- Pause / resume with timer
- Keyboard navigation (arrow keys + number input)
- Responsive design
- Accessible (ARIA roles, live regions, screen reader support)

## Architecture

The project is structured around five classes with clear separation of concerns:

| Class    | Responsibility                                                                                   |
| -------- | ------------------------------------------------------------------------------------------------ |
| `Cell`   | Represents a single cell: value, fixed state, pencilmarks, error state                           |
| `Board`  | 9×9 grid of cells. Exposes `getRow`, `getColumn`, `getBox` and `isCellValid`                     |
| `Solver` | Backtracking algorithm for generating complete boards and creating puzzles with unique solutions |
| `Game`   | Game state: move validation against solution, error counting, undo history                       |
| `UI`     | DOM rendering, event handling, timer, accessibility announcements                                |

### Puzzle generation

1. A complete, valid board is generated by running the backtracking solver on an empty grid with randomized number order (Fisher-Yates shuffle).
2. Cells are removed one at a time. After each removal, the solver checks that the puzzle still has exactly one solution — if not, the cell is restored.
